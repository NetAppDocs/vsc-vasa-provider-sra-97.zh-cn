= VSC和VASA Provider服务会在高度扩展的环境中重新启动
:allow-uri-read: 
:icons: font
:imagesdir: ../media/




== 问题描述

VSC、VASA Provider和SRA虚拟设备可能无法在高度扩展的环境中以最佳性能运行、您可能会注意到VSC和VASA Provider服务频繁重新启动等问题。



== 更正操作

修改VSC、VASA Provider和SRA虚拟设备的RAM和堆内存要求。
