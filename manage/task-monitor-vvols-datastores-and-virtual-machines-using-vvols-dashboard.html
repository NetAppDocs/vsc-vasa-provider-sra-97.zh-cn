ONTAP жҸҗдҫӣзҡ„IOPSж•°жҚ®е°ҶиҝӣиЎҢеӣӣиҲҚдә”е…Ҙе№¶жҳҫзӨәеңЁVVOLдҝЎжҒҜжқҝдёҠгҖӮONTAP жҸҗдҫӣзҡ„е®һйҷ…IOPSеҖјдёҺVVOLдҝЎжҒҜжқҝдёҠжҳҫзӨәзҡ„IOPSеҖјд№Ӣй—ҙеҸҜиғҪеӯҳеңЁе·®ејӮгҖӮ

  • еҰӮжһңжӮЁжҳҜйҰ–ж¬ЎжіЁеҶҢOnCommand APIжңҚеҠЎгҖҒеҲҷеҸӘиғҪеңЁ15еҲ°30еҲҶй’ҹеҗҺеңЁVVOLдҝЎжҒҜжқҝдёҠжҹҘзңӢSAN VVOLж•°жҚ®еӯҳеӮЁеә“зҡ„жүҖжңүжҖ§иғҪжҢҮж Үж•°жҚ®гҖӮ

  • VVOLдҝЎжҒҜжқҝж•°жҚ®жҜҸ10еҲҶй’ҹе®ҡжңҹеҲ·ж–°дёҖж¬ЎгҖӮ

  • еҰӮжһңжӮЁе·ІеңЁ vCenter Server е®һдҫӢдёӯж·»еҠ пјҢдҝ®ж”№жҲ–еҲ йҷӨеӯҳеӮЁзі»з»ҹпјҢеҲҷеҸҜиғҪеңЁдёҖж®өж—¶й—ҙеҶ…дёҚдјҡжіЁж„ҸеҲ° VVol дҝЎжҒҜжқҝдёҠзҡ„ж•°жҚ®еҸ‘з”ҹд»»дҪ•жӣҙж”№гҖӮ

    иҝҷжҳҜеӣ дёә OnCommand API жңҚеҠЎйңҖиҰҒдёҖдәӣж—¶й—ҙжүҚиғҪд»Һ ONTAP иҺ·еҸ–жӣҙж–°зҡ„жҢҮж ҮгҖӮ

  • VVolдҝЎжҒҜжқҝзҡ„*жҰӮиҝ°* portletдёӯжҳҫзӨәзҡ„жҖ»IOPSеҖјдёҚжҳҜиҜ»еҸ–IOPSеҖје’ҢеҶҷе…ҘIOPSеҖјзҡ„зҙҜз§ҜеҖјгҖӮ
